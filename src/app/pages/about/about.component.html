About

This website was created by crmyers for the Software Engineering course (CS3733) at Worcester Polytechnic Institute. If for some reason you have a burning desire to know how this thing works, check out the source code on GitHub, and feel free to submit pull requests! This site was created both for Software Engineering and for myself, as a self-education effort.
