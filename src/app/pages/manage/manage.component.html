Submit New API

Use the form below to add a base API entry. You'll need a name for it, the term & and year it was made in, and the letter of the team that created it. Once the entry is created, you can add a description, image preview, and submit a usable version above!

Important: API names are case sensitive, and names can only be used once. When you submit an API, we convert it into an artifact ID by removing the spaces, so something like "API Update Checker" becomes APIUpdateChecker. Furthermore, since we don't allow an API's files to ever be completely purged, once you've used a name with a given term/year/team combo, you can never re-use it!
